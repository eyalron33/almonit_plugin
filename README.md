# almonit_plugin
ENS+IPFS Firefox plugin by Almonit

# installation
In Firefox, enter `about:debugging`.

Choose `Load Temporary Add-on...`

From the folder of the plugin, choose `manifest.json`

# usage
Navigate to ENS+IPFS websites such as:
- almonit.eth
- pepesza.eth
- portalnetwork.eth
